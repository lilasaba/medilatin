\begin{textblock}{7}(0,3)
  \LHead{What is the problem with Latin speech recognition?}

  \begin{itemize}
      \item Latin is not spoken natively 
      \item There is no available speech database, and it is resource-heavy to create one
      \item Many variants/dialects exists, and we can only make guesses about the pronunciation
      \item The pronunciation mainly depends on
      \begin{itemize}
	      \item the \textbf{era} of the read text 
	      \item the \textbf{georaphical region} where the text originates from
	      \item the \textbf{native language} of the speaker
      \end{itemize}
  \end{itemize}

\end{textblock}

% AM.
\begin{textblock}{2}(4,12.5)
   \LHead{Acoustic model}

\begin{itemize}
\item Mel-Frequency Cepstrum $+$ Energy features were used with Linear Discriminant Analysis (LDA) + Maximum Likelihood Linear Transformation (MLLT), with a splice context of $\pm4$ frames, 10 ms of frame shift.
\item $9\times40$ dimensional spliced up feature vectors served as input to the feed-forward, 6 hidden-layer neural network with p-norm~\cite{kaldi} activation function.
\item Prior to DNN training, a Gaussian Mixture Model (GMM) pre-training was performed.
\item Clustering and Regression Tree (CART)~\cite{kaldi} was applied to obtain acrossâ€’word context dependent shared state phone (or graph) models and their time alignment.
\item The number of senones (and so the size of the DNN softmax output layer) was between 7.000 and 11.000 depending on the nature of the training data.
\item The size of the hidden layers was kept constantly on 2.000.
\item A minibatch size of 512, an initial learning rate of 0.1, and final learning rate of 0.01 was applied in 20 epochs using the Kaldi toolkit~\cite{kaldi}.
\end{itemize}

\end{textblock}
